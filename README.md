# DATN Social App

Instagram-like social network project built with:

- Frontend: React + Vite + TypeScript
- Backend: Node.js + Express
- Database: MongoDB
- Realtime: Socket.IO

## Project structure

- `social-frontend`: web client
- `social-backend`: API, realtime, uploads

## Local development

Frontend:

```bash
cd social-frontend
npm install
npm run dev
```

Backend:

```bash
cd social-backend
npm install
npm run dev
```

## Deployment

See `DEPLOYMENT.md` for the recommended production setup:

- frontend on Vercel
- backend on Render or Railway
- MongoDB on Atlas
